FixMyArea is a full-stack MERN (MongoDB, Express, React, Node.js) civic issue reporting platform that enables users to report and track local community problems. Users can submit issues with images, descriptions, location details, and priority levels. The system provides a personalized dashboard where users can monitor the status of their reports.

Administrators have access to a separate dashboard where they can view all submitted issues, update issue statuses (Pending, In Progress, Resolved), and manage community reports effectively.

The application also includes secure authentication using JWT, profile image uploads with Cloudinary, and an automatic token refresh system to maintain seamless user sessions.
